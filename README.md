# ☁️ OpenNebula Cloud Lab with Security Analysis

## 📌 Project Overview
This project demonstrates the deployment and management of a private cloud environment using OpenNebula. It covers core Infrastructure-as-a-Service (IaaS) concepts including virtualization, networking, storage, automation, and orchestration.

In addition to standard cloud operations, this project integrates a security-focused approach by analyzing potential vulnerabilities, applying hardening techniques, and evaluating access control mechanisms.

---

## 🎯 Objectives
- Deploy and configure OpenNebula cloud environment
- Manage virtual machines and templates
- Configure virtual networking and test connectivity
- Implement storage and datastore management
- Explore OpenNebula Marketplace appliances
- Deploy multi-VM services and Kubernetes cluster
- Apply user roles, quotas, and access control
- Automate operations using CLI and APIs
- Analyze security risks and apply hardening techniques
- Monitor performance and resource utilization
